# 🌼 The Twelve Great Vows of Medicine Buddha

**藥師琉璃光如來十二大願**
**སངས་རྒྱས་སྨན་ལྷའི་སྨོན་ལམ་ཆེན་པོ་བཅུ་གཉིས།**

> *May all sentient beings be free from suffering and find happiness.* 🙏

## About

This page presents the **Twelve Great Vows of Bhaiṣajyaguru (Medicine Buddha)** in three languages:

- 🟡 **中文** — Classical Chinese (Traditional)
- 🔵 **English** — Full translation
- 🟢 **Tibetan** — Original Tibetan script (དབུ་ཅན།)

The Medicine Buddha (藥師琉璃光如來, Sangye Menla) made these twelve great vows before attaining Buddhahood, pledging to benefit all sentient beings who hear his name and sincerely practice his teachings.
